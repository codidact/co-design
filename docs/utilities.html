Utilities—atomic classes for design precision

Co-Design uses "atomic classes" to allow you to specifically modify the presentation of individual elements.

You should be able to create most layouts with these utilities, however some patterns are often repeated. For these, we offer components. You can, though, override the components with these atomic classes.

Click on any of the tabs above, in order to show the utilities for that design area.
